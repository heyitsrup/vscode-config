# dotfiles
My config files for my development tools

![VS Code screenshot](https://github.com/heyitsrup/dotfiles/blob/main/screenshot.png)
```
.
├── .clang-format → Config for clang-format, defining C/C++ code style rules (indentation, braces, spacing)
├── .editorconfig → Editor-agnostic config enforcing consistent coding styles (charset, indentation, line endings) across IDEs
├── .zshrc → Zsh shell startup script; sets aliases, environment variables, and prompt behavior for your terminal
├── extensions.json → Recommended extensions for VS Code
├── README.md → YOU ARE HERE
├── screenshot.png → Screenshot of VS Code
├── settings.json → User config settings for VS Code
└── starship.toml → Configuration file for [Starship](https://starship.rs/), a customizable cross-shell prompt (controls prompt appearance/modules)
```

### Fonts
I use [JetBrains 0xNerd Font](https://www.nerdfonts.com/font-downloads) in editor and 0xNerd Font Mono for terminal.
